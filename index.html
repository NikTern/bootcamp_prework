CSS

  • 'touch style.css' to create style css file in current directory via command prompt
  • Link a CSS text file to the html file by putting in the head: "link rel="stylesheet" href="./assets/style.css"
  • The class elements defined in html can be modified with CSS by typing: .'classname' { \property: value \ property: value \} 
    •

    There are three ways to style a webpage using CSS:

    Inline CSS

  • When a developer wants to style an element directly in the HTML file
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
  • Useful for quick and permanent changes but are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change.

    • E.g. using the style attribute to change the colour of a single heading

    Internal CSS Stylesheet

  • It is also possible to embed an entire stylesheet directly in the HTML file using a >style< element in the >head< element.
  • The internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code.

    • E.g. using the style attribute to change the colour of a single heading

    External CSS Stylesheet

    Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • The 'padding' property adds space around the content inside an element. When you increase padding, you are actually making the entire element bigger! That is because the padding, while transparent, is located inside the element, whereas the margin is the transparent buffer outside an element.

  • Every HTML element is essentially a box. Each box consists of margins outside the element, and the element's borders, padding, and content. This is sometimes referred to as the CSS box model

Javascript

  • 'touch script.js to create new js file'
  • variable assignment (includes semicolon): var variablename = "string"/[arrayitem1, arrayitem2];
  • console log to output a message to the web console (inspect element), useful for testing code: console.log(topics);
    •
  • Javascript and HTML reads top to bottom, so put Javascript at bottom of HTML doc and define variables in order
  • Therefore, unlike linking CSS, link the js file at the bottom of the html document between '/footer' and '/body'
  • To link the JS file, use: >script< src="./assets/script.js">/script<
  • The >script< tag is used to embed a client-side script (JavaScript) into a html document
  • The >script< element either contains scripting statements, or it points to an external script file through the src attribute.
  • This does the same job as the element for CSS. It applies the JavaScript to the page, so it can have an effect on the HTML (along with the CSS, and anything else on the page).
